add
作成者：山内
----------------------------------------------------------------------
目的:
与えられた2つの整数の合計を計算する。

使用方法:
int result = add(int a, int b);

引数:
- a: 第一引数の整数
- b: 第二引数の整数

戻り値:
aとbの合計を返す整数。

使用例:
int sum = add(5, 3); // sumは8になります。
