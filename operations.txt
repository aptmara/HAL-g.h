----------------------------------------------------------------------
InitialSetup
----------------------------------------------------------------------
プロトタイプ

void InitialSetup();

説明

コンソールの初期設定を行います。
文字サイズと文字色・背景色をデフォルト値に設定します。

----------------------------------------------------------------------
PrintAtPosition
----------------------------------------------------------------------
プロトタイプ

void PrintAtPosition(const char* str, int x, bool endline);

説明

文字列 str を指定した x 座標に表示し、endline が true の場合は改行します。
カーソル位置は関数 gotoxy で設定します。

----------------------------------------------------------------------
GetMouseXPosition
----------------------------------------------------------------------
プロトタイプ

int GetMouseXPosition();

説明

現在のマウスの x 座標を取得し、コンソール内の文字数に変換して返します。

----------------------------------------------------------------------
GetMouseYPosition
----------------------------------------------------------------------
プロトタイプ

int GetMouseYPosition();

説明

現在のマウスの y 座標を取得し、コンソール内の文字数に変換して返します。

----------------------------------------------------------------------
ReadCharAt
----------------------------------------------------------------------
プロトタイプ

char ReadCharAt(int x, int y);

説明

指定した座標に存在する文字を読み取り、返します。
x と y の座標で指定された位置にある文字を返します。

----------------------------------------------------------------------
SetColor
----------------------------------------------------------------------
プロトタイプ

void SetColor(int startX, int startY, int endX, int endY, int color);

説明

指定した座標範囲の文字色と背景色を設定します。
範囲は左上の startX と startY から右下の endX と endY までです。
color には設定する文字色・背景色を指定します。
